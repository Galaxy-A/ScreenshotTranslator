# ScreenshotTranslator
## 项目结构
```
ScreenshotTranslator/
├── src/                   # 源代码目录
│   ├── main.py            # 程序入口
│   ├── app.py             # 主应用程序
│   ├── ocr_engine.py      # OCR 引擎
│   ├── result_window.py   # 结果窗口
│   ├── screen_capture.py  # 屏幕捕获
│   ├── settings_window.py # 设置窗口
│   ├── translation.py     # 翻译模块
│   ├── settings.json      # 配置文件
│   └── ocr_result.txt     # 示例结果
├── build.py               # 打包脚本
├── requirements.txt       # 依赖列表
├── Test.bat               # 运行命令
└── README.md              # 项目文档
```
安装使用直接查看运行Test.bat
